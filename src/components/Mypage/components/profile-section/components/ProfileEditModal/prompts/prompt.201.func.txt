조건-API) 프로필 수정 (닉네임)
  - 엔드포인트: POST /api/me/update (AUTH_ENDPOINTS.ME_UPDATE)
  - 설명: 로그인한 사용자의 닉네임(및 선택 시 이메일) 수정
  - 요청: MeUpdateRequest { nickname: string, email?: string }
  - 응답: MeResponse (수정된 프로필)
  - 에러: 400, 401, 409 등

조건-API) 프로필 이미지 업로드
  - 엔드포인트: POST /api/me/profile-image (AUTH_ENDPOINTS.ME_PROFILE_IMAGE)
  - 설명: multipart/form-data로 프로필 이미지 업로드 (jpeg/png/webp, 최대 5MB)
  - 요청: FormData 필드명 'file'
  - 응답: ProfileImageUploadResponse { profileImageUrl?: string, url?: string }
  - 에러: 400(파일 검증 실패), 401 등

조건-파일경로) 구현될 컴포넌트: src/components/Mypage/components/profile-section/components/ProfileEditModal/index.tsx
조건-파일경로) CSS: src/components/Mypage/components/profile-section/components/ProfileEditModal/styles.module.css
조건-파일경로) API: src/commons/apis/auth/me-update.ts (updateMe), src/commons/apis/auth/me-profile-image.ts (uploadProfileImage)
조건-파일경로) 훅: src/components/Mypage/components/profile-section/hooks/useUpdateProfile.ts, useUploadProfileImage.ts
조건-파일경로) 타입: src/commons/apis/auth/types.ts (MeResponse, MeUpdateRequest, ProfileImageUploadResponse)
